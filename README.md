# Data Warehouse Lab

This repository contains hands-on notes and examples for **dimensional data modeling** used in analytics systems (OLAP).

It covers core concepts commonly asked in data engineering and backend interviews.

## Topics

- [Star Schema](./dimensional-modeling/star-schema/README.md)
- [Snowflake Schema](./dimensional-modeling/snowflake-schema/README.md)
- [Slowly Changing Dimensions (SCD)](./slowly-changing-dimensions/README.md)
- [Fact Table Types](./fact-table-types/README.md)
- [Partitioning](./partitioning/README.md)

## Goal

To understand how analytical databases are designed and how they differ from transactional systems (OLTP).

Each section includes:
- Explanation
- Example schemas
- Trade-offs
